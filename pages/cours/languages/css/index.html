Section 1 : Introduction à CSS

Dans cette première section, nous allons plonger dans le monde de CSS (Cascading Style Sheets) et comprendre son rôle essentiel dans la conception web moderne. CSS est un langage de style qui permet de définir l'apparence et la mise en page des pages web.

Le langage CSS s'inscrit dans le cadre du web sémantique, où chaque élément de la page est marqué et structuré de manière significative. Cela signifie que chaque balise HTML a un rôle précis à jouer, ce qui facilite la compréhension du contenu par les navigateurs et les moteurs de recherche. Grâce à CSS, nous pouvons ajouter de la beauté et de la cohérence à notre contenu tout en maintenant la structure sémantique de notre page.

Dans cette section, nous commencerons par explorer les bases de CSS, en abordant notamment la façon de cibler des éléments spécifiques, d'appliquer des styles et de gérer la mise en page. Vous découvrirez également comment créer des règles CSS pour différentes tailles d'écran grâce aux media queries, ce qui vous permettra d'optimiser l'expérience utilisateur sur divers appareils.

Sélecteurs CSS

Les sélecteurs CSS sont des motifs utilisés pour sélectionner des éléments HTML auxquels vous souhaitez appliquer des styles. Ils jouent un rôle essentiel dans la mise en page et la conception web.

Sélecteur d'Élément

Le sélecteur d'élément cible tous les éléments d'un type particulier. Par exemple, pour cibler tous les paragraphes, vous pouvez utiliser :

p {
                    /* Vos styles ici */
                }

Sélecteur de Classe

Le sélecteur de classe permet de cibler des éléments ayant une classe spécifique. Voici comment vous pouvez l'utiliser :

.ma-classe {
                    /* Vos styles ici */
                }

Sélecteur d'ID

Le sélecteur d'ID cible un élément unique avec un ID spécifique :

#mon-id {
                    /* Vos styles ici */
                }

Styles pour le Balisage Sémantique

Appliquons ces sélecteurs CSS à notre balisage sémantique :

h1 {
                    color: #ff6b6b; /* Couleur rouge pour les titres */
                }
                
                .section-title {
                    font-size: 24px; /* Taille de police pour les titres de section */
                }
                
                #section1 {
                    background-color: #f9f9f9; /* Fond gris pour la section 1 */
                }

Propriétés CSS

Les propriétés CSS sont des paramètres qui définissent l'apparence et la mise en page des éléments HTML. Chaque propriété a une valeur associée qui spécifie comment l'élément doit être stylé.

Propriété "color"

La propriété "color" est utilisée pour définir la couleur du texte. Voici un exemple :

color: red;

Propriété "font-size"

La propriété "font-size" contrôle la taille de la police. Vous pouvez utiliser différentes unités de mesure :

font-size: 16px;

Propriété "background-color"

La propriété "background-color" définit la couleur de fond d'un élément. Par exemple :

background-color: #f9f9f9;

Utilisation de CSS avec le Balisage Sémantique

Appliquons ces propriétés CSS à notre balisage sémantique :

h1 {
                    color: red;
                }
                
                .section-title {
                    font-size: 24px;
                }
                
                #section1 {
                    background-color: #f9f9f9;
                }

Une notion fondamentale en CSS est le modèle de boîte (ou modèle de bloc). Chaque élément HTML est considéré comme une boîte rectangulaire dans CSS, et le modèle de boîte définit comment cette boîte est affichée, dimensionnée et espacée.

Les éléments de base du modèle de boîte CSS

Le modèle de boîte CSS se compose de plusieurs éléments clés :

  • Sélecteur HTML : C'est l'élément HTML que vous souhaitez styler, tel qu'un <div> ou un <p>.
  • Propriétés CSS : Ce sont les règles que vous définissez pour l'apparence de l'élément, comme la couleur, la taille de la police, la marge, etc.
  • Contenu : C'est le texte ou les éléments à l'intérieur de la boîte, tels que du texte, des images, etc.
  • Rembourrage (padding) : C'est l'espace entre le contenu et la bordure de la boîte.
  • Bordure (border) : C'est la ligne qui entoure la boîte et sépare le contenu du rembourrage.
  • Marge (margin) : C'est l'espace entre la bordure de la boîte et les autres éléments de la page.
  • Largeur (width) et hauteur (height) : Ce sont les dimensions de la boîte.

Exemple de modèle de boîte CSS

Imaginons que nous voulons styler un paragraphe (<p>) :

p {
                    color: blue;
                    font-size: 16px;
                    padding: 10px;
                    border: 1px solid #ddd;
                    margin: 20px;
                    width: 300px;
                    height: 150px;

Dans cet exemple, nous appliquons des propriétés CSS au paragraphe (<p>). Le texte à l'intérieur de la boîte sera en bleu avec une taille de police de 16 pixels. Il y aura un rembourrage de 10 pixels entre le contenu et la bordure, une bordure de 1 pixel solide de couleur grise, et une marge de 20 pixels autour de la boîte. La largeur de la boîte est de 300 pixels et la hauteur est de 150 pixels.

Boîtes imbriquées

Il est important de noter que les boîtes peuvent être imbriquées les unes dans les autres. Par exemple, vous pouvez avoir une <div> contenant un <p>. Chaque boîte a son propre modèle de boîte et ses propres propriétés CSS.

div {
                    width: 300px;
                    height: 150px;
                    border: 1px solid #ddd;
                }

                p {
                color: blue;
                font-size: 16px;
                padding: 10px;
                margin: 20px;
                }

Dans cet exemple, nous avons un modèle de boîte pour la <div> et un autre modèle de boîte pour le <p> à l'intérieur de la <div>.

Positionnement en CSS (Layout)

Le positionnement en CSS est l'art de contrôler l'agencement et la disposition des éléments HTML sur une page web. Il existe plusieurs techniques de positionnement, chacune ayant ses propres avantages et cas d'utilisation.

Positionnement de base : le flux normal

Par défaut, les éléments HTML suivent le flux normal du document, qui est un flux en bloc. Cela signifie que les éléments sont empilés les uns sur les autres dans l'ordre où ils apparaissent dans le code HTML.

Positionnement avancé : Flexbox

Flexbox est une technique de positionnement puissante qui permet de créer des mises en page flexibles et réactives. Elle est particulièrement utile pour aligner des éléments horizontalement ou verticalement.

.container {
        display: flex;
        justify-content: center;
        align-items: center;
    }

    .item {
        flex: 1;
    }

Positionnement avancé : CSS Grid

CSS Grid est une technique de positionnement bidimensionnelle qui permet de créer des grilles complexes de manière efficace. Elle est idéale pour les mises en page de type tableau.

.grid-container {
        display: grid;
        grid-template-columns: repeat(3, 1fr);
        grid-gap: 20px;
    }

    .grid-item {
        grid-column: span 2;
        grid-row: 1 / 3;
    }
